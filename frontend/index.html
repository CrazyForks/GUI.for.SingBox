WebView2 Runtime Error

Your system's WebView2 Runtime version is not compatible with this software.
Please follow the steps below to fix the issue:

  1. Go to https://developer.microsoft.com/en-us/Microsoft-edge/webview2 and download the fixed version.
  2. Create a data/WebView2 folder in the same directory as the software.
  3. Place the downloaded .cab file into this folder.
  4. Restart the application.
